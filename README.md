# PassLLQP — Alberta LLQP Exam Prep

## Files
- `index.html` — Landing page (home)
- `chapter1.html` — Chapter 1 quiz (free)
- `mock.html` — Mock exam (50 questions)
- `vercel.json` — Routing config for Vercel

## Deployment
1. Upload all files to a GitHub repository
2. Connect the repo to Vercel
3. Vercel auto-deploys to passllqp.vercel.app
